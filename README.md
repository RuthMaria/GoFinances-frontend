### :small_orange_diamond: Commands
- Clone this repository

```
git clone https://github.com/RuthMaria/GoFinances-frontend.git
```
- Install dependencies

```
yarn
```

- Run the project

```
yarn start
```

- Run the tests

```
yarn test
```
